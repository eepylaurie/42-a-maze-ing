# A-Maze-ing

## Description

A-Maze-ing is a maze generator written in Python.
It takes a configuration file,
generates a perfect maze using a recursive backtracking algorithm,
writes it to a file in hexadecimal format,
and displays it visually in the terminal using curses.  
The maze includes a hidden "42" pattern, a BFS-based solver,
and an interactive display with color themes and animated path visualisation.
